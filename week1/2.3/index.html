Waarom noem je de eerste HTML-pagina; index.html?

het is de standaardnaam voor de startpagina van een website omdat webservers automatisch naar dit bestand zoeken in de rootmap.

Met welke HTML versie werken we nu?

We werken momenteel met HTML5

Met welke CSS versie werken we nu?

We werken momenteel met CSS3

Hoe kan het dat een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Een website kan er anders uitzien in verschillende browsers door variaties in hoe elke browser CSS en HTML interpreteert en implementeert

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populairste browser is momenteel Google Chrome, die ongeveer 10-15% van het RAM-geheugen op een gemiddelde computer kan gebruiken
