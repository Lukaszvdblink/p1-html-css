Vragen javascript

1 Het plaatsen van een script in de head vertraagt het laden van de pagina omdat de browser eerst het script moet verwerken. Aan het einde van de HTML laden zorgt ervoor dat de pagina-inhoud eerst zichtbaar is, wat de laadtijd voor de gebruiker verbetert.

2 Het defer-attribuut zorgt ervoor dat het script asynchroon laadt, maar pas wordt uitgevoerd nadat de HTML volledig is geladen. Hierdoor kan de pagina sneller weergegeven worden zonder dat het script de weergave blokkeert.

3HTML structureert de inhoud van een webpagina, zoals tekst, afbeeldingen en links. JavaScript daarentegen voegt interactiviteit en functionaliteit toe, zoals het reageren op gebruikersacties of het dynamisch aanpassen van de inhoud.
